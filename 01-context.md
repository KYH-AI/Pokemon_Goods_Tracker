# 1. 작업 컨텍스트

## 1.1 배경 및 목적

포켓몬 굿즈(카드, 레고, 콜라보 상품)에 투자하는 지인 그룹(10명+, 비개발자)에게 시세와 이벤트 정보를 한눈에 제공하는 정적 웹사이트를 구축한다. GitHub Pages로 배포하고, GitHub Actions로 하루 2회 자동 갱신한다.

## 1.2 범위

| 포함 | 제외 |
|------|------|
| 포켓몬 카드 시세 (최신 확장팩 + 콜라보 이벤트 카드) | 실시간 알림(카카오톡, 텔레그램 등) |
| **국내판/해외판 에디션 구분** | 사용자 로그인/개인화 기능 |
| **PSA/BGS 그레이딩 등급별 시세** (미그레이딩 포함) | 거래/매매 기능 |
| **국내 중고거래 플랫폼 시세 수집** (당근, 번개장터) — 차단 시 폴백 | PSA/BGS 외 그레이딩 회사 (CGC 등) |
| 포켓몬 레고 가격 | |
| 오프라인 이벤트/행사 정보 | |
| 카드샵 행사 / 콜라보 행사 정보 | |

## 1.3 입출력 정의

**입력 (데이터 소스)**

| 카테고리 | 소스 | 수집 방식 | 비고 |
|----------|------|----------|------|
| 카드 시세 (해외판) | TCGPlayer | API/크롤링 | 영문 기반, 해외판 시세 데이터 풍부 |
| 카드 시세 (해외판) | PriceCharting | 크롤링 | 시세 추이, 그레이딩 등급별 가격 (PSA/BGS) |
| 카드 시세 (해외 실거래) | **eBay** | API (Browse API) / 크롤링 | **영어판(en) 전용**. Sold listings(낙찰가) 기반 실거래가. 한국판은 eBay 거래 데이터 없음 |
| 카드 시세 (국내판) | 네이버 카페 시세글 | 크롤링 | **공개 게시판만** (로그인 없음). 한국판 카드 시세 (로그인 필요 시세글은 수집 불가, 데이터 제한적) |
| 카드 시세 (국내 실거래) | **번개장터** | 웹 크롤링 | 키워드 검색 기반, 실거래 매물 가격 수집 (⚠️ ToS 리스크) |
| 카드 시세 (국내 실거래) | **당근마켓** | 웹 크롤링 | 인기매물/검색 페이지 기반 (⚠️ 지역 제한 + 차단 리스크 높음) |
| 그레이딩 시세 | PSA 인증 데이터 (via PriceCharting) | 크롤링 | PSA 10/9/8 등급별 가격 (주로 영어판 데이터) |
| 그레이딩 시세 | BGS 인증 데이터 (via PriceCharting) | 크롤링 | BGS 10/9.5/9/8.5 등급별 가격 (주로 영어판 데이터) |
| 그레이딩 시세 | **eBay Sold listings** | API/크롤링 | **영어판(en) 전용**. 그레이딩 카드 실거래 낙찰가 (PSA/BGS 등급 키워드 검색) |
| 레고 가격 | LEGO 공식 사이트 | 크롤링 | 정가 기준 |
| 레고 시세 | BrickEconomy | 크롤링 | **v1 기본 소스**. 중고 시세, 프리미엄%, 투자 지표 |
| 레고 시세 | BrickLink | API (v2) | 중고 시세, 거래 데이터 (**v2 예정**, v1에서는 제외) |
| 이벤트 | 포켓몬코리아 공식 사이트 | 크롤링 | 공식 행사, 신제품 |
| 행사 | 네이버 카페 (포켓몬 커뮤니티) | 크롤링 | 카드샵 행사, 커뮤니티 정보 (**공개 게시판만**) |
| 환율 | 한국은행 공개 API | API | USD → KRW 환율. 일 1회 갱신, 캐시 활용 |

> ⚠️ **네이버 카페 크롤링 제약**: 로그인 환경 없음. **공개 게시판(비로그인 접근 가능한 게시글)만 수집**.
> 카페 크롤링은 **이벤트/행사 정보**와 **국내판 카드 시세** 두 가지 목적으로 사용되나, 시세글 대부분은 로그인 필요 권한 제어로 인해 공개 데이터가 제한적임.
> 대안 전략: (1) 공개 게시판만 대상 (로그인 없이 접근 가능한 글만), (2) 크롤링 불가 시 수동 입력(구글시트) 폴백 구조 설계

> ⚠️ **당근마켓/번개장터 크롤링 리스크**: 두 플랫폼 모두 자동화 수집을 ToS에서 금지.
> - **번개장터**: 웹 버전(m.bunjang.co.kr)이 있어 크롤링 자체는 가능하나, 안전결제 수수료(6%+) 때문에 표시가격 ≠ 실거래가
> - **당근마켓**: GPS 기반이라 지역별 반복 크롤링 필요. 인기매물 페이지(`/hot_articles`)가 진입점
> - **공통**: GitHub Actions 고정 IP에서 반복 요청 시 차단 가능성 높음 → User-Agent 로테이션, 요청 간격 랜덤화 적용. 차단 시 자동 폴백

> **eBay 수집 방식**: eBay는 공식 Browse API(무료 티어)를 제공하여 Sold listings 조회가 가능.
> API 키 발급 필요 (eBay Developer Program 무료 가입). API 불가 시 `ebay.com/sch/` 검색 결과 + `LH_Complete=1&LH_Sold=1` 파라미터로 낙찰 내역 크롤링 폴백.
> eBay 데이터의 강점: **실제 거래 완료 가격(낙찰가)**이므로 시세 신뢰도가 가장 높음. 특히 PSA/BGS 그레이딩 카드의 거래량이 풍부함.
> **⚠️ eBay는 영어판(en) 카드 전용**: 한국판(kr) 카드는 eBay 거래 데이터가 거의 없으므로 eBay 수집은 영어판 에디션에만 적용한다.

> **한국은행 공개 API**: USD → KRW 환율은 한국은행 Open API(https://ecos.bok.or.kr/api/)를 사용. 무료 사용 가능, API 키 발급 필요. 하루 1회 갱신 후 `exchange_rate.json`에 캐시.

**출력**

| 산출물 | 형식 | 위치 |
|--------|------|------|
| 정적 웹사이트 | HTML/CSS/JS | GitHub Pages (`gh-pages` 브랜치) |
| 수집 데이터 | JSON | `/data/*.json` (리포지토리 내) |
| 수집 로그 | Markdown/JSON | `/logs/` |
| 추적 대상 목록 | JSON (설정 파일) | `/config/watchlist.json` |

## 1.4 제약조건

- **호스팅**: GitHub Pages (정적 사이트만 가능, 서버 사이드 로직 불가)
- **자동화**: GitHub Actions 무료 플랜 (월 2,000분, 스토리지 500MB)
- **크롤링**: robots.txt 확인 필수. 국내 플랫폼(당근/번개장터)은 ToS 위반 리스크를 인지한 상태에서 시도하되, 차단 시 자동 폴백. 요청 간격 준수 필수
- **사용자**: 비개발자 10명+, 모바일 접근 비율 높을 것으로 예상
- **비용**: 무료 운영 원칙 (유료 API 사용 시 사전 합의)
- **언어**: 웹사이트 UI는 한국어, 데이터 소스는 영문 포함 (카드명 매핑 필요)

## 1.5 용어 정의

| 용어 | 설명 |
|------|------|
| **확장팩** | 포켓몬 TCG의 카드 세트 단위 (예: "스칼렛 & 바이올렛 - 흑염의 지배자") |
| **콜라보 카드** | 특별 이벤트/협업으로 발행된 카드 (예: 반고흐 피카츄) |
| **에디션 (edition)** | 카드의 발매 언어/지역 구분. `kr` (한국판), `en` (영어판), `jp` (일본판) |
| **PSA** | Professional Sports Authenticator. 카드 그레이딩 회사. 1~10 등급 (10이 최고) |
| **BGS** | Beckett Grading Services. 카드 그레이딩 회사. 1~10 등급 (10 Black Label이 최고, 0.5 단위) |
| **미그레이딩 (raw)** | 그레이딩 인증을 받지 않은 카드. 가장 기본 상태 |
| **Watchlist** | 시세를 추적할 카드/레고 목록을 정의한 설정 파일 |
| **BrickLink** | 레고 중고 거래 및 시세 확인 플랫폼 |
| **Market Price** | TCGPlayer/BrickLink 등에서 제공하는 시장 평균가 |
