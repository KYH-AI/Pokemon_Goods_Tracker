# 8. 향후 확장 가능성 (현재 범위 밖)

## v2 우선순위 높음

- **가격 히스토리 누적 저장 + 추이 차트** (현재 v1은 최신 스냅샷만 표시. `/data/history/YYYY-MM-DD.json` 형태로 일별 스냅샷 보관 후 최근 7일/30일 차트 표시)
- **일본판(jp) 에디션 시세 추적** (Mercari Japan, Yahoo Japan 등 수집 방법 조사 필요. 콜라보 카드 원본이 일본판인 경우 많음)
- **BrickLink API v2 연동** (현재 BrickEconomy 크롤링만. BrickLink API로 더 정확한 레고 중고 시세 확보)
- **네이버 검색 API 폴백** (현재 API 키 없어 제외. 발급 후 이벤트/시세 수집 정확도 향상)

## v2 우선순위 보통

- 카카오톡/텔레그램 알림 연동 (가격 급등락 시)
- PWA 지원 (오프라인 접근, 홈 화면 추가)
- 추가 그레이딩 회사 지원 (CGC, ACE 등)
- 그레이딩 등급별 가격 추이 차트 (투자 수익률 분석)

## v3 이상

- 구글시트 연동으로 watchlist 관리 UI 제공 (비개발자가 직접 카드 추가 가능)
